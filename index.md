# ctktemplate

Shared `pkgdown` template package for <https://christophertkenny.com> R
packages.

## Use

In `_pkgdown.yml`:

``` yaml
template:
  package: ctktemplate
```

In `DESCRIPTION`:

``` text
Config/Needs/website: christopherkenny/ctktemplate
```

This package provides the shared files in `inst/pkgdown/`.
